== Description ==

Users Birthday Email plugin automatically send an email to WordPress users on their birthday.

This is very easy to use and user friendly interface for non technical person.

It provides a date-picker to collect the user's birthday data. Also you can integrate it with any kind of membership plugin.

Users Birthday Email plugin gives you many filter hooks so that you can modify it easily.

== Installation ==

MODERN WAY:

1. Go to the WordPress Dashboard “Add New Plugin” section.

2. Search For “Users Birthday Email”.

3. Install, then Activate it.


OLD WAY:

1. Upload the plugin folder to the /wp-content/plugins/ directory.

2. Activate the plugin through the ‘Plugins’ menu in WordPress.


== Frequently Asked Questions ==

= Does this work without the membership plugin? =

Yes, it does not require any membership plugin to use. Because it provides a date-picker to collect the user's birthday data.

= Can this plugin work with any membership plugin data? =

Yes, it can work with any membership plugin data. Because this plugin has many filter hooks that can be easily integrated with other membership plugins.
